# AIML-task-3
# Student Performance Predictor

## Task 3: Evaluate and Make Predictions

### Objective

Evaluate the trained Linear Regression model and make predictions.

### Tools

- Python
- Pandas
- scikit-learn

### Steps

1. Create the dataset.
2. Split into training and testing sets.
3. Train the Linear Regression model.
4. Predict scores on the test set.
5. Calculate Mean Absolute Error (MAE).
6. Compare actual and predicted values.
7. Predict the score for 4.5 study hours.

### Output

- Mean Absolute Error
- Actual vs Predicted table
- Predicted score for 4.5 study hours
